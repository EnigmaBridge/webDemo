Scalability

Enigma Bridge service utilizes a large number of secure processing elements. We allocate these processors to users on-the-fly to adjust to changing user requirements in real time.

Simplicity

Use of Enigma Bridge is simple. You can use it with only a general ability to develop software. All configuration is done automatically behind the scenes.

High Availability

Enigma Bridge service comprises at least two separate servers with automatic fail-over and load-balancing. A large number of secure processors makes resiliency an inherent property of Enigma Bridge.
